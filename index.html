Weisfeiler-Lehman(WL) test

  • Graph Isomorphism Problem (NP-hard): Identify whether two graphs are topologically identical.
  • Implementation (Necessary)
    • Aggregates labels of nodes and their neighborhoods
    • Hash the aggregated labels to unique new labels

Representation Power

DEF 1 (Maximum power): A maximally powerful GNN maps two nodes to the same locations only if they have identical subtree structures with identical features on the corresponding nodes.

Remark 1: The recursiveness of WL test implies the question whether a GNN maps two neighborhoods (or two multisets) to the same embedding or representation.

*Remark 2: A maximally powerful GNN would **never* map two different neighborhoods to the same representation, *injection*.

Lemma 2 (Countability of Range of GNN): Assume the input feature space $\mathcal X$ is countable. Let $g^{(k)}$ be the function parameterized by a GNN’s $k$-th layer for $k=1,\dots,L$, where $g^{(1)}$ is defined on multisets $X\subset \mathcal X$ of bounded size. Graph Isomorphism Network (GIN)

Lemma 3 (Decomposition Uniqueness): Assume $\mathcal X$ is countable. There exists a function $f:\mathcal X\rightarrow \mathbb R^n$ so that $h(X)=\sum_{x\in X}f(x)$ is unique for each multiset $X\subset \mathcal X$ of bounded size. Moreover, any multiset function $g$ can be decomposed as $g(X)=\phi\left(\sum_{x\in X}f(x)\right)$.

Corollary 1 - Assume $\mathcal X$ is countable. There exists $f:\mathcal X\rightarrow \mathbb R^n$ so that for $\forall$ irrational number $\epsilon$ such that $h(c,X)=(1+\epsilon)f(c)+\sum_{x\in\mathcal X}f(x)$ is unique. Moreover, any function over the pair $(c,X)$ can be decomposed as $g(c,X)=\varphi((1+\epsilon)f(c)+\sum_{x\in\mathcal X}f(x))$ for some function $\varphi$.

Less Powerful GNNs

1-layer perceptron

Lemma 4 (Breakup of Injectivity) - There $\exists X_1\neq X_2$ that are finite multisets so that for any linear mapping $W$ $\sum_{x_1\in X_1}\mathrm{ReLU}(Wx_1)=\sum_{x_2\in X_2}\mathrm{ReLU}(Wx_2)$ holds.

The breakup results from the lack of universal approximation which is possessed by MLPs.

  • Even if it can distinguish different graphs (Injectivity), it might not be able to detect the similarity between them.

Mean- and max-pooling

  • Mean (distribution), Max (representative)

Corollary 2 (Distribution of mean-pooling) - Assume $\mathcal X$ is countable. There $\exists f:\mathcal X\rightarrow \mathbb R^n$ so that for $h(X)=\frac{1}{|X|}\sum_{x\in\mathcal X}f(x)$, $f(X_1)=f(X_2)$ if and only if $X_1$ and $X_2$ have the same distribution. That is, $\exists (S,m_0)$ such that $X_1=(S,k_1m_0)$ and $X_2=(S,k_2m_0)$ for some $k_1,k_2\in\mathbb N_+$.

Corollary 3 (Underlying set of max-pooling) - Assume $\mathcal X$ is countable. Then $\exists f:\mathcal X\rightarrow \mathbb R^n$ so that for $h(X)=\max_{x\in X}f(x)$, $h(X_1)=h(X_2)$ if and only if $X_1$ and $X_2$ have the same underlying set.`

Some doubts that I have met [with Answers]

  • Q1 - What is the term “well-defined” defined?

    • A1 - A function is well-defined, meaning it is not ambiguous so that each element of its domain is corresponding to a unique value.

  • Q2 - What is the difference between bijection and isomorphism?

    • A2 - There are two cases that should be considered. When it comes to two sets, without structures (operations), they are identical. However, the isomorphism is usually used for representing the preservation of structures. For example, two groups $A$ and $B$ are isomorphic so that $\exists g:A\rightarrow B$ such that $g(a_1)g(a_2)=g(a_1a_2)$ where $a_1,a_2\in A$.

  • Q3 - How does mean aggregator work for functions defined on sets rather than on multisets?

    • A3 - Similar to the construction introduced in this paper, we can prove $\exists f:\mathcal X\rightarrow \mathbb R^d$ so that $\forall X\subset\mathcal X$, $h(X)=\frac{1}{|X|}\sum_{x\in X}f(x)$ is unique. Let us consider $f(x)=N^{-Z(x)}$ where $Z:\mathcal X\rightarrow \mathbb Z_+$ and $N\in\mathbb Z_+$ with sufficiently large $N$. For $X_1\neq X_2$, assume $h(X_1)=h(X_2)$. Thus,This equation can be rewritten by $X_0=X_1\cup X_2$ asThen, $\frac{\mathbb 1(x_1\in X_1)}{|X_1|}-\frac{\mathbb 1(x_2\in X_2)}{|X_2|}\in\left\{\frac{1}{|X_1|},-\frac{1}{|X_2|},\frac{1}{|X_1|}-\frac{1}{|X_2|} \right\} \subset \left[ -\frac{1}{|X_2|}N^{-Z(x)},\frac{1}{|X_1|}N^{-Z(x)} \right]$. The sufficiently large $N$ guarantees that $\frac{\mathbb 1(x\in X_1)}{|X_1|}-\frac{\mathbb 1(x\in X_2)}{|X_2|}=0$, implying $\mathbb 1(x\in X_1)=\mathbb 1(x\in X_2)$ and $|X_1|=|X_2|$. Therefore, $X_1=X_2$, leading to the injectivity. But it is not for multisets.

  • Q4 - What types of functions are called universal?

    • A4 - A universal function is a function that can imitate all other functions.
  • Q5 - Why the features of the earlier iterations may generalize better?
    • A5 - The deeper iterations might lead to overfitting such that the inherent discrimination cannot be distinguished really well. Therefore, including features from earlier iterations can improve the generalization performance.
